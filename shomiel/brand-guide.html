07 — Motion

Motion

Motion is purposeful and restrained. Use faster durations for feedback, slower curves for reveals. Hover the boxes below to preview each easing curve.

--ease-fast 150ms ease
--ease-base 250ms ease-out
--ease-spring 400ms cubic-bezier(0.34, 1.56, 0.64, 1)
--ease-smooth 600ms cubic-bezier(0.23, 1, 0.32, 1)

Reduce motion: All animations check prefers-reduced-motion. When the user has set this preference, transitions are disabled globally via the animations.css override.

08 — Rules

Brand guidelines

These guardrails protect brand consistency across all marketing, product, and partner touchpoints.

Category Do ✓ Don't ✗
Logo Use the SVG at any scale. Keep clear space equal to the mark's width on all sides. Stretch, rotate, apply drop shadows, change colors, or use on cluttered backgrounds.
Color Use navy + gold as primary brand colors. Use semantic colors (success/error) for status only. Use brand gold for error states. Use gradient fills on the logo. Add glow effects.
Typography Use Space Grotesk for display headings and Inter for body. Load fonts from Google Fonts. Use system fonts for headings. Mix more than 3 weights in a single context. Use <h1> more than once per page.
Photography Use clean, documentary-style photography with natural light and people in context. Use stock photos with obvious posing. Apply filters or color overlays that clash with navy/gold.
Writing Be precise and direct. Lead with outcomes. Use active voice. Short paragraphs. Use jargon to impress. Write walls of text. Make promises that can't be kept in a headline.
Motion Use motion to guide attention and confirm actions. Keep animations under 600ms. Add decorative animations that don't serve a purpose. Animate without checking reduced-motion.
Accessibility Maintain WCAG AA contrast (4.5:1 for text). Provide alt text. Support keyboard navigation. Rely on color alone to convey meaning. Remove focus outlines. Use icon-only buttons without labels.
